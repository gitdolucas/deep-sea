# Tower Upgrade System

## Overview

Each defense evolves visually and mechanically across **three tiers**. The upgrade feels like the tower is growing from the ocean floor — coral encrusting it, crystals forming, the structure becoming more alive and powerful.

## Upgrade Philosophy

| Level | Theme                        | Description                                              |
|-------|------------------------------|----------------------------------------------------------|
| L1    | Raw / Functional             | Built from basic ocean stone and coral. Dim glow.        |
| L2    | Reinforced / Crystal         | Reinforced with bioluminescent crystal. Wider base, more detail, brighter emission. |
| L3    | Fully Awakened               | Pulses with Tideheart energy. Animated glow, larger radius, distinct silhouette. |

## Cost Curve

| Upgrade   | Cost              |
|-----------|-------------------|
| Build L1  | Base cost          |
| L1 -> L2  | 2x base cost      |
| L2 -> L3  | 4x base cost      |
| **Total** | **7x base cost**  |

This creates a **strategic tension**: fully upgrading one tower (7x) vs. building two new L1 towers (2x) vs. one L2 + one L1 (3x).

## Visual Upgrade Cues

These are consistent across all tower types:

| Cue              | L1                  | L2                    | L3                           |
|------------------|---------------------|-----------------------|------------------------------|
| Size             | Compact             | Medium                | Imposing                     |
| Glow color       | Dim teal            | Vivid cyan            | White-hot core + colored aura |
| Radius ring      | Not visible         | Appears on ground     | Pulses on ground             |
| Idle animation   | Static              | Slow pulse            | Full breathing glow cycle    |

## Upgrade Animation

When a tower is upgraded:
1. Bright flash at the tower's position
2. Ring of bubbles bursts outward
3. Tower transforms (brief morph/grow animation)
4. New radius ring fades in
5. Tower settles into its new idle animation

## Tower Progression Summary

| Tower           | L1                        | L2                          | L3                                    |
|-----------------|---------------------------|-----------------------------|---------------------------------------|
| Tideheart Laser | Single dim beam, short range | Dual beam, medium range     | Triple rotating beam, long range, pierces |
| Bubble Shotgun  | 3-bubble spread, slow fire | 5-bubble spread, faster     | 7-bubble burst, AOE pop on impact     |
| Vibration Zone  | Small slow radius, mild   | Medium radius, stronger slow | Large radius, slow + damage over time |
| Current Cannon  | Pushes Stonecrabs only    | Pushes all except Colossus  | Knocks back + stuns briefly           |
| Ink Veil        | Small blind cloud, short  | Wider cloud, longer duration | Persistent ink storm, reduces armor   |
| Arc Spine       | Chains 2 targets          | Chains 4 targets            | Chains 6 targets, burn DoT            |

## Salvage

Players can sell any tower to reclaim **50% of total shells invested** (including upgrade costs). This allows repositioning defenses as the wave composition changes.
