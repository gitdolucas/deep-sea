# Bubble Shotgun

**Type:** AOE / Spread Cone
**Best Against:** Stoneclaw

## Description

Pressurized saltwater blasts in a spread cone. Each bubble bursts on contact, dealing area damage. Ideal against Stoneclaw floods.

## Upgrade Tiers

| Level | Visual                          | Spread | Behavior                                |
|-------|----------------------------------|--------|-----------------------------------------|
| L1    | Small coral nozzle               | 3 bubbles | Slow fire rate, narrow spread        |
| L2    | Wider barrel, barnacle details   | 5 bubbles | Faster fire rate, wider spread       |
| L3    | Multi-barrel cannon, glowing     | 7 bubbles | Rapid burst, AOE pop on impact       |

## Sprite Specs

### Level 1
- Simple coral tube mounted on a rock base
- Faint blue bubble particles on fire
- Static idle

### Level 2
- Reinforced barrel with bioluminescent crystal accents
- Visible radius ring on ground
- Slow pulse idle

### Level 3
- Triple-barrel cannon with glowing pressure chambers
- Bright cyan aura, pulsing radius ring
- Bubbles constantly forming at barrel tips

## Build Cost

| Action   | Cost (Shells) |
|----------|---------------|
| Build    | Base cost     |
| L1 -> L2 | 2x base cost |
| L2 -> L3 | 4x base cost |

## Strategic Notes

- Primary counter to Stoneclaw swarms — spread damage clears groups
- At L3, bubbles pop with AOE splash, hitting nearby enemies
- Less effective against single fast targets (Razoreel slips through the spread)
